Exercise 04 — Using JupyterLab with the Course Python Environment (Formative00 Part 4 Standard)

Goal: Open and run an existing .ipynb notebook using the standardized course environment created in Formative00 Part 4.

Important: Do not install Jupyter with pip for this exercise. You must use Miniforge + Conda and launch JupyterLab from your course environment. This prevents “wrong Python / wrong pip” problems.

Step 1 — Download the notebook from the MIS202_ReadOnly OneDrive folder available in D2L

  1. Open the course shared OneDrive read-only folder provided by your instructor.
  2. Download: MIS202Ex4.ipynb
  3. Save it to your local notebook folder:

C:\Users\<StarID>\Temp\WSU_MIS\notebooks

Step 2 — Launch JupyterLab from the correct environment

  1. Open Miniforge Prompt.
  2. Activate the base environment:
conda activate wsu_mis_base
  1. Start JupyterLab:
jupyter lab

Step 3 — Open and run the notebook

  1. In JupyterLab, open MIS202Ex4.ipynb from:

C:\Users\<StarID>\Temp\WSU_MIS\notebooks

  1. Confirm the notebook kernel is WSU MIS Base (wsu_mis_base) (top-right of JupyterLab).
  2. Run the notebook cells using Shift + Enter.
  3. Follow the notebook directions exactly. (Order matters.)

Reminder: Notebooks keep “state.” If you change code in an earlier cell but do not re-run it, later cells will still use the old value.

Step 4 — Shut down correctly

  1. In JupyterLab, save your notebook (Ctrl+S).
  2. Close the browser tab.
  3. In the Miniforge Prompt window, press Ctrl + C and then type y to shut down the server.
